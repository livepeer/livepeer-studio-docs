# Playback a video

This guide provides information about working with livestream playbacks and working with a video player to display your video.

## Step 1: Get the `playbackId` of an existing asset

Get the `playbackId` of an existing asset. An asset's `playbackId` can be found using the dashboard,
or by making an API request using a `stream.id` or `asset.id`. If you're making an API request,
you can find the `playbackId` in the response object of the [stream](https://docs.livepeer.studio/reference/api/get-stream#response)
or [asset](https://docs.livepeer.studio/reference/api/get-assets#response-example) request.

If you haven't created an asset yet, follow the 
[upload a video asset guide](/guides/on-demand/upload-video-asset#api) to do so.

<CodeBlockTabs>
<CodeBlockTabItem value="Node" label="Node">

```js
// Livestream

const getPlaybackId = await fetch("https://livepeer.studio/api/stream/{id}", {
    method: "GET",
    headers: {
        Authorization: `Bearer ${process.env.API_TOKEN}`,
        "Content-Type": "application/json",
    },
});

// On Demand

const getPlaybackId = await fetch("https://livepeer.studio/api/asset/{id}", {
    method: "GET",
    headers: {
        Authorization: `Bearer ${process.env.API_TOKEN}`,
        "Content-Type": "application/json",
    },
});
```

## Step 2: Embed a video player

To build an application that plays your livestream or On Demand assets, you need to use a video 
player compatible with the HLS protocol. 

We recommend using Livepeer's inhouse [livepeer.js](https://livepeerjs.org/)
player as it makes building with Livepeer effortless. There are also many HLS video players out there, for example 
[video.js](https://videojs.com/) or [JW Player](https://www.jwplayer.com/) which either have built-in 
support or have plugins to support HLS.

<CodeBlockTabs>
<CodeBlockTabItem value="livepeer.js" label="LivepeerClient.js">

```js
import { LivepeerConfig, createReactClient } from '@livepeer/react';
import { studioProvider } from 'livepeer/providers/studio';
 
const livepeerClient = createReactClient({
  provider: studioProvider({
    apiKey: process.env.NEXT_PUBLIC_STUDIO_API_KEY,
  }),
});
 
// Pass client to React Context Provider
function App() {
  return (
    <LivepeerConfig client={livepeerClient}>
      <Livepeer />
    </LivepeerConfig>
  );
}
```

</CodeBlockTabItem>
<CodeBlockTabItem value="livepeer.js video player" label="VideoPlayer.js">

```js
import { VideoPlayer } from '@livepeer/react';
 
const playbackId = await getPlaybackId.json().playbackId
 
function Livepeer() {
  return (
    <VideoPlayer
      autoplay
      loop
      muted
      playbackId={playbackId}
    />
  );
}
```
