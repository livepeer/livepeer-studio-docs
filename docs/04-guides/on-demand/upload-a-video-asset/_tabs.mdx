# Upload a video asset

This guide provides you the instructions to upload a video from the
[Livepeer Studio Dashboard](./dashboard) or through the [Livepeer API](./api).

Throughout this guide you'll see a few terms mentioned multiple times, asset and
task:

- **Asset**: an object that represents a file
- **Task**: an action taken on the file
