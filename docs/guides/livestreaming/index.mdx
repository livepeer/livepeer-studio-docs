Documentation in this section is organized in order of topics you may find on your journey to build livestreaming applications:

## Introducing Livepeer Livestreaming

- You can integrate your streaming application directly with the Livepeer Studio API.

- You can use the Livepeer Studio Dashboard to livestream without writing any
  code.
